Characterization extraction rule:
For each characterization method, extract not only the method itself, but also the main finding or conclusion explicitly supported by that method whenever stated in the text.

This includes:
- what the method measures
- what structural / compositional / mechanistic conclusion it supports
- which sample the characterization result belongs to

Examples:
- XRD -> supports phase identification, peak weakening/disappearance, alloying, interdiffusion, dominant phase conclusions
- ICP -> supports elemental ratio conclusions, trace-element conclusions, non-equiatomic composition conclusions
- Cole-Cole analysis -> supports dielectric relaxation conclusions
- Electron microscopy -> supports particle size, aggregation, dense morphology, dispersion conclusions

Characterization-supported finding rule:
If a characterization method is mentioned together with an explicit analytical conclusion, do not stop at extracting only the method name.
Also extract the supported finding as either:
- a StructureConcept
- a LossMechanism
- a Limitation
- or an ExperimentalSample attribute / remark
depending on the context.

Characterization-supported binding rule:
When a method provides evidence for a structure, phase, composition, mechanism, or conclusion, connect:
- the corresponding sample to the CharacterizationMethod
- the CharacterizationMethod to the supported finding using supports_finding

Preferred patterns:
- ExperimentalSample -> characterized_by -> CharacterizationMethod
- CharacterizationMethod -> supports_finding -> StructureConcept / LossMechanism / Limitation
