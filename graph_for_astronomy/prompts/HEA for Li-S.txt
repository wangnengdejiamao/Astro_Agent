   - `"entity_types"`

2. **Triple Format**  
   All triples MUST be a 3-element list in the exact format:
   `[subject, relation, object]`  

   - All three elements MUST be strings  
   - No nested lists, no missing elements  

   Examples:  
   - Correct: `["TiO2", "adsorbs", "Li2S6"]`

   All attributes MUST be strings in the exact format:

   `"key: value"`

   - `"areal_capacity: 6.2 mAh/cm²"`  

   Do NOT use numbers alone.  
   Do NOT repeat the entity name or schema type as an attribute value.

4. **Entity Consistency (Very Important)**  
   - Entity names must be IDENTICAL across:
     - `"attributes"` keys  
     - all `"triples"`
