# 通用视觉规范（每页都套用）

> 本文件供执行人参考。每个 `slides_prompts/*.md` 都已把这些规则内嵌到 prompt 里，但如果模型出图偏离，请把本段贴回去再生一次。

- **画幅**：16:9，建议 1920×1080；纯白底 `#FFFFFF`，零边距出血。
- **主题色**：薄荷绿 `#12CCB9`，即 `RGB(18, 204, 185)`；不要 teal、不要 cyan。
- **辅助色**：深灰 `#2F3A45`（正文）、浅灰 `#ECEFF3`（卡片底）、强调红 `#FF6B6B`（仅 Before / 痛点）。
- **字体**：中文**思源黑体**；英文 **Inter** 或 **Helvetica Neue**；标题粗体，正文常规。
- **标题样式**：左上角，主标题中文 36–44pt 加粗；前置 8×64px 薄荷色短竖条；其下方一行英文副标题，浅灰 18pt 斜体。
- **页脚**：右下角小字 `林佳茂 · 中山大学物理与天文学院 · 2026`，上方一根 1px 薄荷色横线。
- **构图灵魂**：**禁止**两栏 / 三栏并列。整套 PPT 至少轮换：① 流程箭头横向、② 分支树、③ 同心圆、④ Z 字流、⑤ 时间轴、⑥ 偏轴卡片、⑦ 蜂窝模块墙。
- **信息密度**：每页 ≤ 5 个文字块，每块 ≤ 25 字；要点全部图形化。
- **Icon**：线性描边 2px，圆角，薄荷色描边，白底。**不要**彩色实心、emoji。
- **数字强化**：关键指标用「特大薄荷色数字 + 小灰色单位 + 一行解释」三段式。
- **不要**生成虚假 Logo、不要拼错英文术语、不要在图里加无关装饰星空 / 宇宙 / 星云背景。
